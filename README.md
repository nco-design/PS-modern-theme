## Description

This theme was designed with the Trimui Smart Pro S in mind. It should work on other trimUI devices and devices running SpruceOS<br>
Most of the icons come from [Google Font](https://fonts.google.com/icons)
Other icons were drawn by me, taking inspiration from PlayStation icons (Vita and PSP).

## How to install
### Installing themes

Download the release as a .zip<br>
Unzip it on your computer, then take the folder and paste it on your SD card in the /theme folder.<br>

### Installing boot logos

Download the release as a .zip<br>
Unzip it on your computer, then take the folder and paste it on your SD card in the /App/Bootlogo/Imgs folder.<br>
